# 🎉 实验完成！

你已经成功完成了赋值与局部变量的实验，掌握了以下核心概念：

| 概念 | 说明 |
|------|------|
| 任意类型赋值 | 规则可以赋予数字、字符串、数组、对象等任意 JSON 类型 |
| 表达式赋值 | `:=` 右侧可以是函数调用、算术运算等任意表达式 |
| 隐式 `true` | 没有 `:=` 赋值的规则条件成立时，自动返回 `true` |
| 局部变量 | 用 `:=` 在规则内部声明，作用域仅限当前规则 |
| 作用域隔离 | 不同规则中的同名局部变量互不影响 |
| 变量不可变 | 一旦赋值不能更改，需要使用新的变量名 |

## 关键要点

- **Rego 是声明式的** —— 变量代表一个确定的值，不是可变的存储位置
- **隐式 true 让代码更简洁** —— 布尔类型的规则可以省略 `:= true`
- **局部变量提升可读性** —— 用有意义的局部变量名拆解复杂的表达式链
