# 恭喜完成本章实验！

你已经掌握了 Rego 中的 `every` 全量迭代：

- **AllOf 语义**：`every x in collection { ... }` 确保集合中每一个成员都满足所有条件
- **多条件 every**：花括号中的多个条件是逻辑与关系，每个成员必须同时满足
- **every 中实现逻辑或**：
  - **in 操作符**：`item.color in {"red", "blue"}` 简洁表达"属于集合"
  - **辅助函数**：通过函数重载实现复杂的逻辑或条件
  - **蕴含模式**：`¬P ∨ Q` 表达"如果 P 则 Q"的约束
- **some + every 组合**：先用生成表达式筛选子集，再对子集做全量检查，逻辑更直观

**关键要点**：`some` 实现 AnyOf（存在一个满足即可），`every` 实现 AllOf（所有都必须满足）。两者结合可以表达丰富的策略逻辑。
