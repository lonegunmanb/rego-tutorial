# 常见陷阱（下）：引用与迭代

本次实验你将学习 Rego 中与引用和迭代相关的常见陷阱：

1. **未声明迭代变量陷阱**：变量名可能与包级变量冲突，导致意外行为
2. **无效路径引用陷阱**：缺失字段导致规则静默失败，以及 **exists** 函数和 **not x == x** 技巧
3. **错别字与输入数据预校验**：拼写错误不会报错，以及如何编写输入预校验
4. **反转含迭代规则的陷阱**：**!=** 不等于"不包含"
