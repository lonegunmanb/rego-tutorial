# 🎉 实验完成！

你已经成功完成了 Rego 的第一个实验，掌握了以下核心概念：

| 概念 | 说明 |
|------|------|
| `package` | 声明代码所属的命名空间 |
| `import rego.v1` | 使用 Rego v1 语法 |
| `:=` 赋值 | 定义规则（全局变量）的值 |
| `if {}` 条件赋值 | 只有条件成立时才进行赋值 |
| `default` | 为规则提供默认值，确保始终有输出 |
| `deny contains msg if` | conftest 使用的策略检查模式 |

## 关键要点

- **规则就是全局变量** —— Rego 中的规则本质上就是全局变量，所有规则的值汇总成输出文档
- **条件不成立 ≠ false** —— 条件不满足时，变量不会被定义（未定义），而不是被赋值为 `false`
- **`default` 让输出确定** —— 使用 `default` 可以确保变量始终有值，方便调用方处理
