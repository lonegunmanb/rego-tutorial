# 🎉 实验完成！

你已经成功完成了 Rego 逻辑运算符的实验，掌握了以下核心概念：

| 概念 | 说明 |
|------|------|
| 逻辑与（AND） | 规则体内的多个条件按顺序书写，它们之间就是逻辑与关系 |
| 逻辑或（OR） | 声明多个同名规则，任意一条成立即为成立（并联电路） |
| 不可赋予不同的值 | 同名规则在不同分支成立时，赋予的值必须相同，否则触发 `eval_conflict_error` |
| 逻辑非（NOT） | 使用 `not` 关键字反转条件的真假 |

## 关键要点

- **串联电路 = 逻辑与** —— 规则体 `if { ... }` 中的多个条件全部成立，规则才成立
- **并联电路 = 逻辑或** —— 同名的多条规则，任意一条成立，变量就会被赋值
- **`not` 反转真假** —— 特别适合检测字段缺失（`not input.xxx`）
- **值必须一致** —— 同名规则赋予的值必须相同，否则会报冲突错误
